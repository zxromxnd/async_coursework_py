# Асинхронна Система Обробки та Моніторингу Задач

Курсова робота з основ програмування та компонентів програмної інженерії.

**Автор:** Брюховецький Євген, група ІМ-51

## Опис проєкту

Система для виконання задач з пріоритетами в асинхронному режимі.

## Основні компоненти

- Черга з пріоритетами
- Асинхронна обробка задач
- Кешування результатів
- Event-driven комунікація
- API authentication
- Система логування

## Технології

- Python >= 3.8
- Асинхронне програмування
- Streams та async iterators
- Декоратори
